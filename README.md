# 🚀 ResumeIQ AI

> AI-powered Resume Intelligence Platform built with FastAPI, React, LangChain, RAG, PostgreSQL, Docker, and Google Gemini.

---

## 📌 Overview

ResumeIQ AI is an AI-powered resume analysis platform that helps job seekers optimize their resumes for Applicant Tracking Systems (ATS).

Users can upload resumes, compare them against job descriptions, receive AI-generated feedback, improve ATS scores, and generate actionable suggestions using Retrieval-Augmented Generation (RAG).

---

## ✨ Features

- 🔐 User Authentication (JWT)
- 📄 Resume Upload (PDF)
- 🧠 AI-powered Resume Analysis
- 🎯 ATS Score Generation
- 💼 Job Description Matching
- 📚 RAG-based Knowledge Retrieval
- 📈 Resume Improvement Suggestions
- 📊 Personal Dashboard
- 📂 Resume History
- ☁️ Cloud Deployment
